# 🚀 Quantum Flip

An interactive browser-based game built with pure HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://quantum-flip-blcr.netlify.app/

---

## 🎮 About the Game

Quantum Flip is a geometry-based puzzle/logic game where players interact with dynamic elements and explore spatial behavior.

* Smooth gameplay experience
* Clean UI with modern styling
* Fully responsive design
* Built from scratch (no frameworks)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 📂 Project Structure

```
Game/
 ├── index.html
 ├── styles.css
 ├── game.js
 ├── levels.js
```

## ⚡ Features

* Interactive gameplay mechanics
* Custom level handling
* Lightweight and fast

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Banka-Lakshmi-Chaitanya-Ram/quantum-flip.git
```

2. Open the project folder

3. Run `index.html` in your browser
